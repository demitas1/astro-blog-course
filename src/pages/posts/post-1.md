## テキストの装飾

**太字テキスト** は `**` で囲みます。

_斜体テキスト_ は `*` で囲みます。

~~取り消し線~~ は `~~` で囲みます。

## リスト

### 順序なしリスト

- 項目1
- 項目2
  - ネストした項目
  - もう一つのネスト
- 項目3

### 順序付きリスト

1. 最初の項目
2. 2番目の項目
3. 3番目の項目

## コード

インラインコード: `const message = "Hello World"`

コードブロック:

```javascript
function greet(name) {
  return `こんにちは、${name}さん！`;
}

console.log(greet("太郎"));
```
